Netflix Clone 🎬

A modern Netflix Clone built using HTML, CSS, JavaScript, Firebase Authentication, and Tailwind CSS.

This project includes:

* Netflix-style UI
* Hero banner section
* Movie rows
* Responsive design
* Firebase Authentication
* Google Login
* Video Player Page
* Movie playback support

⸻

🚀 Features

    🔥 Firebase Setup

1. Create a Firebase Project
2. Enable Authentication
3. Enable:
    * Email/Password Login
    * Google Login
4. Copy Firebase Config
5. Paste it inside:

config.js
